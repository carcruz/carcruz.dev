Smart City Manager is an interactive dashboard to communicate 5G capabilities such as enabling greater public safety, improved citizen services, and real-time monitoring of crowds, traffic, and the environment.

### Role

Front-end engineer. I built this web-based geographic dashboard using d3 and Leaflet. The dashboard consumes real-time data using the MQTT protocol.
